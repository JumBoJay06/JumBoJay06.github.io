程式碼結構與詳細解析

YouBikeApiClient

這部分不用調整，沿用程式碼。

YouBikeStationEntity

  • 將原本的 Model 改用 Lombok(@Data) 來實作 getters, setters, toString, equals, 和 hashCode 方法。
  • @MongoEntity 是 Panache 的注解，用來將這個 Java 類別與 MongoDB 的一個集合（collection）對應起來。
  • 繼承 PanacheMongoEntityBase 來簡化 DB 操作。v
完整檔案

YouBikeStationDTO

給客戶端的欄位，只給客戶端所需要的資料

  • 一樣用 Lombok(@Data) 來實作 getters, setters, toString, equals, 和 hashCode 方法。
完整檔案

YouBikeStationMapper

在 Entity 和 DTO 之間轉換欄位是一件繁瑣且容易出錯的工作，透過 YouBikeStationMapper 自動化轉換。

  • componentModel = "cdi" 讓這個 Mapper 可以被 Quarkus 的依賴注入框架管理。
完整檔案

YouBikeService

整個專案最核心的業務邏輯

  • Panache 透過 Active Record 模式為 Entity 類別直接賦予了資料庫操作的能力。
